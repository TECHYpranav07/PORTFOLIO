PROJECTS

TITLE DESCRIPTION LINK
ROBOTICS CAR The robot car is an interactive vehicle that combines remote and voice control via an Arduino and HC-05 Bluetooth module. It can move forward, backward, left, and right, and even execute complex movement patterns like a square. This versatile functionality makes it an excellent educational tool for exploring robotics, programming, and wireless communication, providing hands-on experience and enhancing learning in these fields.
RADAR SYSTEM Our project involves building a radar system using Arduino, a servo motor, and a distance sensor. The servo motor rotates the sensor, enabling it to scan the surrounding area. The distance sensor measures object distances, and the Arduino processes this data to create a radar-like visualization. The outcome is a simple yet effective method for detecting objects in the vicinity.
SMART PARKING SYSTEM Our Smart Parking System utilizes Arduino, LEDs, and sensors to efficiently manage parking spaces. The sensors detect the presence of vehicles, and corresponding LEDs indicate the availability of parking spots. This system provides real-time information, making parking hassle-free and efficient. The outcome is optimized parking management, reducing congestion and enhancing user convenience.
WATER ALARM SYSTEM For our first Electronics project, we created a water level alarm system. It alerts users when water reaches a critical level in a tank, triggering an alarm. Using Multisim for simulation and KiCad for PCB design, we developed a reliable solution for water level monitoring, enhancing safety and preventing overflow incidents. The outcome was an effective and easily deployable system for various water storage applications.
